🗺 マインドマップ セットアップ

GitHubにデータを保存するため、以下の情報を入力してください。

GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
スコープ: public_repo にチェック

← 一覧に戻る

初回セットアップ手順
  1. github.com でアカウント作成
  2. 右上の「＋」→「New repository」でリポジトリ作成
    （Public・README追加にチェック）
  3. Settings → Pages → Branch: main で GitHub Pages を有効化
  4. Settings → Developer settings → Personal access tokens → Tokens (classic) でトークン作成
    （スコープ: public_repo）
  5. このフォームに入力して「設定を保存してはじめる」をクリック
