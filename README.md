# マインドマップアプリ

Notionに埋め込んで使えるインタラクティブなマインドマップWebアプリです。  
データはこのGitHubリポジトリに保存され、コミット履歴でバージョン管理されます。

## セットアップ

1. このリポジトリをGitHubにアップロード
2. Settings → Pages → Branch: main で GitHub Pages を有効化
3. `https://{username}.github.io/{repo}/setup.html` を開いてセットアップを完了

## 使い方

| 操作 | 方法 |
|------|------|
| ノード追加 | ＋ボタンをクリック |
| テキスト編集 | ノードをダブルクリック |
| ノード移動 | ドラッグ＆ドロップ |
| ノード削除 | ×ボタンをクリック |
| 画面移動 | 背景をドラッグ |
| ズーム | マウスホイール / ズームボタン |
| 保存 | 「保存」ボタン（1保存 = 1コミット） |

## Notionへの埋め込み

1. Notionで `/embed` を入力
2. 各マップの「URLコピー」で取得したURLを貼り付け
3. iframe内でそのまま操作可能
